# Full system overview

From the division of the functions the full system, the time constraints and the set up requirements for this system, a devision of subsystems can be made. The decision is made to implement a simple capacitive sensor circuit for the pitch control and read and process the frequency digitally for clean and highly adaptable sound output. The rest of the inputs are then either sliders, knobs, or buttons, which are read out using a simple ADC or direct digital inputs. The audio output is done using a DAC/audio card and then amplified before the speaker. The visual output is done over HDMI for adaptability and due to time constraints. The playing guide is an array of LEDs, digitally controlled using digital output pins. The full system is illustrated in [](#fig-Design). The Hardware & Sensing subsystem consists of providing the sensing/user input elements, power to the whole system, amplifying the output signal to a speaker and the visualisation. The signal processing subsystem consists of the digital processing done in the micro controller and micro processor, except for the visualisation. The division is also illustrated in [](#fig-Design).
